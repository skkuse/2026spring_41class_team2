# MovieLens Subset Seed 적재 계획

## 목적

MovieLens Subset은 MovieLens 전체 데이터에서 Cinemate가 실제로 서비스할 영화 3,000개를 골라낸 seed 기준 데이터셋이다. 이 subset을 기준으로 영화 카탈로그, 출연진/제작진, 추천/통계 데이터를 서비스 시작 전에 적재한다.

이 문서는 MovieLens subset 기반 seed 적재 계획의 상위 문서다. 세부 선정 기준, 컬럼 매핑, CSV 생성 규칙, 검증 기준은 하위 seed 계획 문서에서 관리한다.

DB 테이블 정의의 기준 문서는 [../../db-schema/README.md](../../db-schema/README.md)를 따른다. MovieLens 주요 파일 설명은 [../../movielens-dataset.md](../../movielens-dataset.md)를 따른다.

## 왜 Subset이 필요한가

MovieLens 원본은 Cinemate가 서비스할 범위보다 크고, 전체 평점/추천 계산 데이터를 그대로 DB에 저장하기에는 용량 부담이 크다. Cinemate는 Supabase Free tier를 사용하므로 서비스에 필요한 영화와 추천 데이터만 선별해 적재해야 한다.

또한 Cinemate의 영화 상세 원천은 TMDB이고, 영화 ID도 TMDB movie id를 기준으로 사용한다. 따라서 MovieLens 전체 영화를 적재하지 않고, MovieLens에서 `tmdbId`가 있고 TMDB 표시 데이터가 충분한 영화만 subset으로 확정한다.

이 subset은 다음 데이터의 공통 기준이다.

- 영화 카탈로그
- 장르 연결
- 출연진/제작진
- 영화별 평점/리뷰 집계 초기값
- 온보딩 기반 Item CF 추천 결과

## Seed 대상 테이블

MovieLens subset 파이프라인에서 미리 적재할 테이블은 다음 10개다.

| 그룹          | 테이블                                       | 세부 문서                                                          |
| ------------- | -------------------------------------------- | ------------------------------------------------------------------ |
| 영화 카탈로그 | `movies`, `genres`, `movie_genres`           | [1-1.movie-catalog-seed-plan.md](./1-1.movie-catalog-seed-plan.md) |
| 출연진/제작진 | `people`, `movie_casts`, `movie_crew`        | [1-2.movie-credits-seed-plan.md](./1-2.movie-credits-seed-plan.md) |
| 추천/통계     | `movie_stats`, `movie_similarities`, `movie_tags`, `movie_tag_relevances` | [1-3.movielens-recommendation-seed-plan.md](./1-3.movielens-recommendation-seed-plan.md) |

## 원천 데이터 역할

| 원천      | 역할                                                       |
| --------- | ---------------------------------------------------------- |
| MovieLens | 후보 선정, MovieLens id 매핑, 평점/태그 집계, Item CF 계산 |
| TMDB      | 영화 상세, 장르, 포스터, 줄거리, 출연진/제작진             |

## Subset 산출물

- 최종 서비스 영화 3,000개
- seed CSV 8개
- 기준 영화별 Item CF 추천 결과
- MovieLens Tag Genome 기반 자연어 추천 후보 데이터

| 1-2  | [1-2.movie-credits-seed-plan.md](./1-2.movie-credits-seed-plan.md) | `people`, `movie_casts`, `movie_crew`        | TMDB credits 기반 인물, 출연진, 제작진 seed 생성        |

## 의존성

```text
movies
  ├─ movie_genres
  ├─ movie_casts
  ├─ movie_crew
  ├─ movie_stats
  │   └─ movie_similarities
  └─ movie_tag_relevances

genres
  └─ movie_genres

people
  ├─ movie_casts
  └─ movie_crew

movie_tags
  └─ movie_tag_relevances
```

`1-2`와 `1-3`은 모두 `1-1`의 `movies` 적재 이후 진행한다. `1-2`와 `1-3` 사이에는 직접 FK 의존성이 없다.
