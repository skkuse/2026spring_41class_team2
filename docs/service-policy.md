# Cinemate 서비스 정책

이 문서는 Cinemate의 서비스 동작 정책을 기록한다.

여기서 정책은 서비스가 특정 상황에서 어떤 동작을 해야 하는지 정한 제품 수준의 규칙을 의미한다. 구현 방식, DB 구조, API 필드, 쿼리 조건 같은 세부 사항은 구현계획 문서와 DB 스키마 문서에서 다룬다.

## 인증 및 사용자 정책

- 비로그인 사용자는 영화 탐색, 검색, 상세 정보 조회만 가능하다.
- 비로그인 사용자는 AI 영화 추천 채팅, 캐릭터 채팅, 맞춤 추천을 사용할 수 없다.
- 비로그인 사용자는 영화 찜 상태를 가질 수 없다.
- 비로그인 사용자가 찜 버튼이나 리뷰 작성 버튼을 누르면 로그인 페이지로 보낸다.

## 온보딩 정책

- 비로그인 온보딩은 허용하지 않는다.
- 온보딩은 회원가입 후 온보딩을 완료하지 않은 로그인 사용자만 접근한다.
- 온보딩 완료 후에는 온보딩 화면으로 다시 넘어가지 않는다.
- 온보딩에서는 장르가 아니라 인기 영화 목록에서 선호 영화 5개를 선택한다.
- 온보딩 영화 선택은 정확히 5개를 선택해야 완료할 수 있다.
- 온보딩 영화 후보는 MovieLens와 TMDB가 연결된 인기 영화로 한정한다.
- 온보딩에서는 영화 검색을 제공하지 않는다.
- 온보딩에서 선택한 영화는 맞춤 추천의 기준 영화로 사용한다.

## 영화 데이터 정책

- 영화 데이터는 MovieLens와 TMDB를 함께 사용한다.
- MovieLens는 추천과 평점 데이터의 기준으로 사용한다.
- TMDB는 영화 상세 정보의 기준으로 사용한다.
- IMDB는 사용하지 않는다.
- 영화 적재 범위는 MovieLens와 TMDB가 연결된 영화만으로 한다.
- TMDB 전체 영화를 적재하지 않는다.
- 영화 동기화는 최초 적재 후 고정으로 한다.
- 검색은 기본적으로 서비스에 적재된 영화 안에서만 제공한다.
- TMDB 실시간 검색은 기본 검색 정책에 포함하지 않는다.

## 영화 정보 정책

- 장르는 TMDB 장르 기준으로 통일한다.
- MovieLens 장르는 화면과 검색의 장르 기준으로 사용하지 않는다.
- 배우와 감독 정보는 TMDB에서 가져온다.
- 출연진과 제작진은 분리해서 관리한다.

## 평점 정책

- 평점은 TMDB 평점을 사용하지 않는다.
- 화면 카드/상세의 기본 평점은 MovieLens 평균 평점과 Cinemate 리뷰 평점을 함께 반영한다.
- 사용자가 작성한 리뷰 평점은 영화의 Cinemate 평점에 반영한다.

## 리뷰 및 찜 정책

- 리뷰는 한 유저가 한 영화에 하나만 작성할 수 있다.
- 리뷰 수정/삭제는 일단 배제한다.
- 리뷰 목록은 공개 조회 가능하다.
- 리뷰 작성과 리뷰 좋아요는 로그인 사용자만 가능하다.
- 찜 기능은 로그인 사용자만 가능하다.

## 맞춤 추천 정책

- 맞춤 추천은 온보딩에서 선택한 영화를 기준으로 제공한다.
- 맞춤 추천은 MovieLens 기반 추천을 사용한다.
- 맞춤 추천 결과에는 온보딩에서 선택한 영화, 이미 찜한 영화, 이미 리뷰한 영화를 포함하지 않는다.
- 추천 결과가 부족하면 평점 높은 영화로 보충한다.
- 맞춤 추천 화면은 기준 영화별 추천 섹션을 제공한다.
- 맞춤 추천 섹션 제목은 `{영화 제목}을 좋아한 사람들이 함께 좋아한 영화` 형식을 사용한다.

## 채팅 정책

- AI 영화 추천 채팅과 캐릭터 채팅은 분리한다.
- AI 영화 추천 채팅은 대화 이력을 유지한다.
- 캐릭터 채팅은 대화 이력을 유지한다.
- 캐릭터 채팅은 AI 영화 추천 채팅과 별도 도메인으로 관리한다.
- 캐릭터 대본/설정은 수동으로 관리한다.
- 캐릭터 데이터는 TMDB 자동 파생 데이터가 아니라 서비스 기획 데이터로 본다.
- 캐릭터는 특정 영화와 연결된다.
- 캐릭터는 필요하면 TMDB 배우 정보와 연결할 수 있다.

## 접근 권한 정책

- 영화, 장르, 배우/인물 정보는 공개 읽기 가능하게 한다.
- 프로필은 본인만 수정 가능하게 한다.
- 찜은 본인 데이터만 접근 가능하게 한다.
- 추천 채팅은 본인 대화만 접근 가능하게 한다.
- 캐릭터 채팅은 본인 대화만 접근 가능하게 한다.
- 리뷰 목록은 공개 조회 가능하게 한다.
- 리뷰 작성과 리뷰 좋아요는 로그인 사용자만 가능하게 한다.
